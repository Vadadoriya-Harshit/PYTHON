
 -----Data type in python--------------
Text Type:	str
Numeric Types:	int, float, complex
Sequence Types:	list, tuple, range
Mapping Type:	dict
Set Types:	set, frozenset
Boolean Type:	bool
Binary Types:	bytes, bytearray, memoryview
None Type:	NoneType

-------list data type----
 ex :
     x = ["Oranges","Grapes","Apples"]

-------Tuple data type----
 ex :
     x = ("Oranges","Grapes","Apples")
